# 🏫 Magic Academy

> 마법학교 교장이 되어 다양한 직군의 학생을 육성하고 학교를 성장시키는 2D 탑다운 캐주얼 경영 게임

## 게임 소개

Magic Academy는 낡은 마법학교의 신임 교장이 된 플레이어가 학생을 선발하고, 수업을 개설하고, 시설을 확장해 학교를 성장시키는 경영 시뮬레이션 게임입니다.

마법사뿐 아니라 기사, 상인, 성직자, 도적 등 다양한 직군을 꿈꾸는 학생들이 입학합니다. 학교생활 중 크고 작은 인카운터가 발생해 학생의 성장을 가속시키거나, 때로는 이탈로 이어집니다. 졸업한 학생들의 활약이 학교 평판으로 되돌아옵니다.

**"내가 키운 영웅이 세상을 바꾼다."**

---

## 핵심 특징

- **10개 직군 계열** — 마법사, 기사, 상인, 성직자, 도적 등 판타지 세계관 내 다양한 직업
- **인카운터 시스템** — 조건부 + 랜덤 혼합 방식으로 학생의 성장, 이탈, 계열 전향이 발생
- **학기 사이클** — 봄 학기 → 여름 방학 → 가을 학기 → 겨울 졸업식 반복
- **졸업생 평판** — 졸업생의 세계 활약이 뉴스로 돌아와 학교 성장에 영향
